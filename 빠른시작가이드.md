# 🚀 빠른 시작 가이드 - 5분 안에 배포하기

## 1단계: GitHub 계정 만들기 (이미 있다면 건너뛰기)
- https://github.com 접속
- "Sign up" 클릭하여 무료 계정 생성

## 2단계: 저장소 만들기
1. GitHub 로그인 후 우측 상단 "+" 버튼 클릭
2. "New repository" 선택
3. Repository name에 `hospital-guide` 입력
4. **Public** 선택 (필수!)
5. "Create repository" 버튼 클릭

## 3단계: 파일 업로드
1. 생성된 저장소 페이지에서 "uploading an existing file" 링크 클릭
2. **hospital-guide 폴더 안의 모든 파일**을 드래그하여 업로드
   - index.html
   - body-parts.html
   - injection-precautions.html
   - non-covered.html
   - bone-injection.html
   - 모든 .png 이미지 파일들
3. 페이지 하단 "Commit changes" 버튼 클릭

## 4단계: GitHub Pages 활성화
1. 저장소 상단의 **"Settings"** 탭 클릭
2. 왼쪽 메뉴에서 **"Pages"** 클릭
3. "Source" 아래:
   - Branch: **main** 선택
   - Folder: **/ (root)** 선택
4. **"Save"** 버튼 클릭

## 5단계: 완료! 🎉
- 약 1-2분 후 페이지 상단에 **초록색 체크마크**와 함께 URL이 나타납니다
- URL 형식: `https://사용자명.github.io/hospital-guide/`
- 이 URL을 복사하여 QR 코드를 만드세요!

## QR 코드 만들기

### 방법 1: 네이버 QR코드 (추천)
1. 네이버에서 "QR코드 생성" 검색
2. 네이버 QR코드 서비스 접속
3. 위에서 복사한 URL 붙여넣기
4. QR 코드 생성 및 다운로드

### 방법 2: 온라인 QR 코드 생성기
- https://www.qr-code-generator.com/
- https://www.the-qrcode-generator.com/

## ❓ 자주 묻는 질문

**Q: 비용이 드나요?**
A: 완전 무료입니다! GitHub Pages는 무료 호스팅 서비스입니다.

**Q: 나중에 이미지를 바꿀 수 있나요?**
A: 네! GitHub 저장소에서 이미지를 삭제하고 새로 업로드하면 됩니다.

**Q: 휴대폰에서도 잘 보이나요?**
A: 네! 모바일 최적화가 되어 있어 스마트폰에서 보기 편합니다.

**Q: URL이 너무 긴데 짧게 만들 수 있나요?**
A: 네! bit.ly나 Naver 단축URL 서비스를 이용하시면 됩니다.

## 🆘 문제가 생겼나요?

### 페이지가 안 보여요
- Settings → Pages에서 "main" 브랜치가 선택되었는지 확인
- 2-3분 정도 기다려보세요
- 브라우저 새로고침 (Ctrl+F5)

### 이미지가 안 보여요
- 모든 .png 파일이 업로드되었는지 확인
- 파일명에 특수문자가 없는지 확인
