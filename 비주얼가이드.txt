## 🆘 문제 해결

### Q: 404 에러가 나요
```
┌─────────────────────────┐
│  404                    │
│  File not found         │
└─────────────────────────┘

A: 2-3분 기다린 후 다시 접속하세요.
   GitHub Pages 배포는 시간이 조금 걸립니다.
```

### Q: 이미지가 안 보여요
```
A: 체크리스트:
   ✅ 모든 .png 파일을 업로드했나요?
   ✅ 파일명에 한글이 포함되어 있나요?
   ✅ 파일 크기가 너무 크지 않나요? (<25MB)
```

### Q: URL을 잃어버렸어요
```
A: 찾는 방법:
   1. GitHub Repository 접속
   2. Settings → Pages 클릭
   3. 상단에 URL 표시됨
```
