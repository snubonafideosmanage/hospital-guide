┌─────────────────────────────────────────────────────────────────────┐
│ 📍 STEP 1: GitHub 계정 만들기 (1분)                                 │
├─────────────────────────────────────────────────────────────────────┤
│ https://github.com → Sign up                                        │
│ 이메일 + 비밀번호 + 사용자명 입력 → 이메일 인증                    │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ 📍 STEP 2: 저장소 만들기 (1분)                                      │
├─────────────────────────────────────────────────────────────────────┤
│ 우측 상단 "+" → New repository                                      │
│ Repository name: hospital-guide                                     │
│ ✅ Public 선택 → Create repository                                  │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ 📍 STEP 3: 파일 업로드 (2분)                                        │
├─────────────────────────────────────────────────────────────────────┤
│ "uploading an existing file" 클릭                                   │
│ hospital-guide 폴더의 모든 파일 드래그 앤 드롭                      │
│ (HTML 파일 5개 + PNG 이미지 14개 = 총 19개)                         │
│ Commit changes 클릭                                                 │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ 📍 STEP 4: 웹사이트 활성화 (1분)                                    │
├─────────────────────────────────────────────────────────────────────┤
│ Settings → Pages → Branch: main → Save                              │
│ 새로고침 (F5) → URL 확인                                            │
│ https://사용자명.github.io/hospital-guide/                          │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ 📍 STEP 5: QR 코드 생성 (1분)                                       │
├─────────────────────────────────────────────────────────────────────┤
│ 네이버 검색 "QR코드 생성" → 첫 번째 결과                            │
│ URL 입력 → QR코드 만들기 → 다운로드                                │
└─────────────────────────────────────────────────────────────────────┘

╔══════════════════════════════════════════════════════════════════════╗
║                         🎯 핵심 포인트                               ║
╠══════════════════════════════════════════════════════════════════════╣
║ ✅ 반드시 Public으로 설정                                           ║
║ ✅ Branch는 "main" 선택                                             ║
║ ✅ 모든 파일(19개)을 한 번에 업로드                                 ║

╚══════════════════════════════════════════════════════════════════════╝
