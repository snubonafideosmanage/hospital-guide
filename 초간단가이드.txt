# 🎯 초간단 배포 가이드 (5분 완성)

## 준비물
- ✅ 컴퓨터
- ✅ 인터넷 연결
- ✅ 다운로드 받은 hospital-guide 폴더

---

## STEP 1: GitHub 가입 (1분)
https://github.com 접속 → Sign up → 이메일/비밀번호/사용자명 입력

> 이미 계정이 있으면 로그인만 하세요!

---

## STEP 2: 저장소 만들기 (1분)

1. 로그인 후 우측 상단 **"+" 버튼** 클릭
2. **"New repository"** 클릭
3. 입력:
   - Repository name: `hospital-guide`
   - **Public 선택** ✅
4. **"Create repository"** 버튼 클릭

---

## STEP 3: 파일 올리기 (2분)

1. **"uploading an existing file"** 클릭
2. hospital-guide 폴더 열기
3. **Ctrl+A** (전체 선택)
4. 파일들을 드래그해서 GitHub 페이지에 놓기
5. 하단 **"Commit changes"** 클릭

---

## STEP 4: 웹사이트 활성화 (1분)

1. 상단 **"Settings"** 클릭
2. 왼쪽 메뉴 **"Pages"** 클릭
3. Branch를 **"main"** 선택
4. **"Save"** 클릭
5. 페이지 새로고침 (F5)

✅ 초록색 박스에 URL이 나타나면 성공!
`https://사용자명.github.io/hospital-guide/`

---

## STEP 5: 모바일로 확인 (30초)

### 스마트폰에서 확인:
위 URL을 스마트폰 브라우저에 입력

### QR 코드 만들기:
1. 네이버에서 "QR코드 생성" 검색
2. URL 붙여넣기
3. 다운로드

---

## ✅ 완료!

이제 QR 코드를 출력해서 병원에 붙이면 됩니다!

---

## 🆘 안 되면?

- **404 에러**: 2-3분 기다리고 다시 접속
- **이미지 안 보임**: 모든 .png 파일을 올렸는지 확인
- **URL 모름**: Settings → Pages에서 확인

---

## 💡 TIP

**나중에 수정하려면?**
1. GitHub에서 파일 클릭
2. 연필 아이콘(수정) 또는 휴지통(삭제)
3. 새 파일은 "Add file" → "Upload files"

**1-2분 후 자동 업데이트!**
